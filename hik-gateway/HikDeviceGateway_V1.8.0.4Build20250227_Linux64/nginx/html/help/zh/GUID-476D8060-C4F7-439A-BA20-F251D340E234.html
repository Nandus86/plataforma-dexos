概述

简介

Device Gateway 是一款集成中间件，你可以添加并管理ISUP5.0和ISAPI门禁设备和ISUP5.0/2.0视频设备，添加设备后可以通过 Device Gateway 测试协议，查看各项功能是否正常使用。支持通过HTTP协议和RTSP协议供三方平台软件集成对接，提供设备配置、操作和码流转发等功能。

运行环境

特性

要求

操作系统

  • Microsoft(R) Windows 10 (64位)、Windows Server 2012 R2 (64位),、Windows Server 2016 (64位) 、 Windows Server 2019 (64-位)

    说明

    对于Windows Server 2012 R2 (64位)，需要提前安装补丁包KB2999226。

  • CentOS 7（64位）

  • Ubuntu 20.04（64位）

  • Red Hat Enterprise Linux9（64位）

CPU

Intel(R) Core(TM) i5-7500 @ 3.0 GHz, 4核及以上

内存

8 GB或以上

网卡

千兆网卡（最新驱动）
说明

如果电脑进入休眠模式，Device Gateway 会异常。请关闭电脑休眠模式以保证软件运行正常。
