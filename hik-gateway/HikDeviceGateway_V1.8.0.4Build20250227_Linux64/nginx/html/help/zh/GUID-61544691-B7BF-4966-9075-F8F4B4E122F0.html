步骤二：添加设备

添加视频设备和门禁设备后，Device Gateway 可通过HTTP协议和RTSP协议供三方平台软件集成对接，完成设备配置、操作和码流转发等功能。

  1. 可选操作： 配置端口映射。在路由器等网络设备上配置端口映射参数后，需在 Device Gateway 配置相同的外部端口号和外部IP地址使端口映射生效。
    1. 单击 网关配置 > 网络 > 端口映射。
    2. 勾选启用，启用平台端口映射或设备端口映射功能。输入对应的外部端口号和外部IP地址，并保存设置。
    说明

    • 默认情况下，保存设置后，Device Gateway 会自动重启。

    • 建议设置ISUP2.0取流端口号为15000-17000。

    图 1 配置端口映射
  2. 添加设备。以添加视频设备为例。

    • 手动添加。

      1. 单击设备管理 > 视频类设备 > 添加。

      2. 选择添加方式和 协议类型，并配置相关参数。

      3. 单击添加。

    • 从预注册列表添加。预注册列表展示当前向网关尝试注册的ISUP5.0视频设备。选中设备，单击添加至设备列表，输入登陆密钥。

    图 2 添加设备
    说明

    若未手动添加设备，且该设备位于预配置列表中，手动添加设备后，该设备将从预注册设备列表中移除。

选择设备进行协议测试。详情查看：步骤三：协议测试。
